# 📊 Gestão de Vendas Dados

Repositório oficial de governança de dados do projeto Gestão de Vendas.

---

## 👥 Controle de Acesso
- Apenas membros autorizados têm acesso ao repositório (privado)
- Alterações nos dados exigem aprovação via Pull Request
- Nenhuma modificação direta na branch `main` é permitida

---

## 🔄 Controle de Versões
- Todas as alterações são registradas via commits com mensagens descritivas
- Branches devem seguir o padrão: `feature/nome-da-mudanca`
- Mudanças só são incorporadas após revisão e aprovação de PR

---

## 🔍 Auditorias
- O histórico completo de commits serve como trilha de auditoria
- Issues no GitHub Projects registram tarefas de auditoria e backup
- Revisões periódicas são feitas a cada entrega de sprint

---

## 📁 Estrutura de Pastas
- `/documentacao` — políticas, regras e documentos oficiais
- `/dados` — arquivos de dados do projeto
- `/fontes` — referências e bibliografias utilizadas
